# SEO Webdesigner

Guide SEO spécialisé pour les webdesigners, UX/UI designers, intégrateurs et studios créatifs. Axé sur le portfolio, le personal branding technique, et la conversion en missions.

## 1. Spécificité SEO du webdesigner

- Double audience : clients **non-techniques** (PME, entrepreneurs) + clients **tech** (startups, développeurs)
- Le site lui-même EST sa carte de visite — performances et design doivent être irréprochables
- Forte concurrence nationale → **niche géographique ou sectorielle indispensable**
- Portfolio = actif SEO principal (pages de cas clients indexables)

## 4. Structure des pages

### Page d'accueil
```markdown
# Webdesigner Freelance — [Nom] | [Ville]

[Site lui-même = démonstration vivante du savoir-faire]

[Positionnement spécifique : pour qui, quoi, comment]
Exemple : "Je crée des sites qui convertissent pour les TPE/PME à [ville]."

## Mes projets récents
[3-4 cas clients avec visuels + résultats]

## Ce que je fais
- Création site vitrine et e-commerce
- Refonte et optimisation UX
- Design UI (Figma)

## Parlons de votre projet
[Formulaire ou Calendly]
```

### Page cas client (très importante pour le SEO)
```markdown
# [Nom Client] — [Type de projet] | [Nom Designer]

**Client :** [secteur, taille]
**Mission :** [création / refonte / UX audit]
**Technologies :** [Next.js, Figma, Webflow...]
**Résultats :** [conversions +X%, vitesse +Y%, etc.]

[Screenshots avant/après]
[Description du processus : brief → wireframes → design → intégration]

[Lien vers le site en ligne si possible]
```

## 7. Stratégie de contenu

| Article | Intention |
|---|---|
| "Combien coûte un site web en 2025 ?" | Forte conversion |
| "WordPress vs Webflow vs Next.js : lequel choisir ?" | Info + positionnement |
| "Pourquoi votre site perd des clients (et comment corriger)" | Douleur client |
| "Étude de cas : refonte du site [secteur]" | Preuve + SEO |
| "Core Web Vitals : guide pour les non-développeurs" | E-E-A-T |

---

## 8. Signaux de conversion

- [ ] Site lui-même rapide et irréprochable (LCP < 1,5s)
- [ ] Portfolio avec 5+ cas clients détaillés
- [ ] Tarifs ou fourchette (PME veut savoir si abordable)
- [ ] Technologies maîtrisées listées
- [ ] Témoignages clients
- [ ] Formulaire de brief (type de projet, budget, délai)
- [ ] Profil Malt lié

---

## 9. Checklist avant mise en ligne

- [ ] Core Web Vitals : LCP < 2,5s, CLS < 0,1 (ironie si le site du designer est lent)
- [ ] Schema ProfessionalService validé
- [ ] Behance et Dribbble liés (sameAs)
- [ ] Malt profil complet
- [ ] 5+ cas clients avec pages dédiées
- [ ] Blog avec 3 articles au lancement
- [ ] Mentions légales + SIRET
