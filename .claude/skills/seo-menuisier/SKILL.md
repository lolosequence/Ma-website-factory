# SEO Menuisier

Guide SEO spécialisé pour les menuisiers, charpentiers, ébénistes et poseurs de cuisines/parquets. Axé sur le référencement local, les certifications, et la conversion en devis.

## 1. Intentions de recherche prioritaires

| Intention | Exemple | Action |
|---|---|---|
| Devis | "menuisier devis gratuit [ville]" | Formulaire |
| Remplacement | "remplacement fenêtres double vitrage [ville]" | Devis |
| Sur mesure | "menuisier sur mesure [ville]" | Contact |
| Aides | "menuisier RGE fenêtres MaPrimeRénov [ville]" | Devis |
| Info | "prix fenêtres PVC alu [ville]" | Contact |

## 5. Certifications clés à mettre en avant

- **RGE (Reconnu Garant de l'Environnement)** → accès aux aides MaPrimeRénov' / CEE
- **Qualibat** → gage de sérieux pour les particuliers
- **PRO DE LA PERFORMANCE ÉNERGÉTIQUE**
- Numéro SIRET + assurance décennale (obligatoire à mentionner)

---

## 6. Galerie avant/après

Les photos de réalisations sont le premier facteur de conversion pour un menuisier :
- Minimum 15-20 réalisations en galerie
- Systématiquement : avant + après + contexte
- Tags alt descriptifs : "cuisine sur mesure chêne naturel [ville]"
- Partager sur Houzz, Habitissimo, Pinterest
